# RAG-OnePieceWiki

## Overview

RAG-OnePieceWiki is a project designed to leverage Retrieval-Augmented Generation (RAG) techniques to provide insightful information about the One Piece universe. This project utilizes a combination of data ingestion, querying, and storage mechanisms to create a comprehensive knowledge base.

## Features
- **Data Ingestion**: Efficiently ingest data from various sources, including Wikipedia and other One Piece-related content.
- **Querying**: Perform complex queries to retrieve relevant information quickly.
- **Storage**: Utilize Qdrant for efficient storage and retrieval of data.

## Installation

To get started with RAG-OnePieceWiki, clone the repository and install the required dependencies:

```bash
git clone https://github.com/yourusername/RAG-OnePieceWiki.git
cd RAG-OnePieceWiki
pip install -r requirements.txt
```

## Usage

After installation, you can run the code:

```bash
python ingest_wiki.py
```

```bash
python query_wiki.py
```
